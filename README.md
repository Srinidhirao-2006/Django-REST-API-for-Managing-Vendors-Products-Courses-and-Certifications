# Vendor Product Course API

Django REST Framework project for managing Vendors, Products, Courses, and Certifications.

## Features
- Vendor CRUD API
- Product CRUD API
- Course CRUD API
- Certification CRUD API
- Vendor-Product Mapping
- Product-Course Mapping
- Course-Certification Mapping
- Filtering APIs
- Swagger Documentation

## Installation

1. Clone the repository
2. Install dependencies

pip install -r requirements.txt

3. Run migrations

python manage.py migrate

4. Run server

python manage.py runserver
